ArUco 2D SLAM

Technologies : Python, OpenCV, NetworkX

It is a vision-based mapping system that processes an image containing ArUco markers to create a simplified map. Each tag is treated as a node, and the distances between them form edges, building a spatial graph of the terrain. It is used to calculate tag positions and relationships, enabling basic SLAM simulation for robotics, navigation, or indoor localization tasks.
